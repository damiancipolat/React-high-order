# Custom field mask component

- This component is a wrapper using recompose of the module "react-input-mask", https://www.npmjs.com/package/react-input-mask.

## Use
To run this project

  ```js
  <InputMask
    maskFormat={"99-9999"}
    maskChar="-"
    maskError="Error writing text"
    validator={validatorTest}
  />
  ```

## Run
To run this project

  ```js
   $ npm install
   $ npm Start
  ```
